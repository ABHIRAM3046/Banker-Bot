# Banker Bot AWS

## Overview
The Banker Bot is an intelligent chatbot built using AWS Lex and other AWS services. It helps users manage their banking needs through natural language conversations.

## Features
- Account balance inquiry
- Fund transfer
- Transaction history
- Customer support

## Technologies Used
- AWS Lex
- AWS Lambda
- AWS DynamoDB
- AWS S3
- Python
